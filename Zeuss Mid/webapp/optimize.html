Как настроить DNS вручную

  1. Wi-Fi: Настройки → Wi-Fi → значок (i) у вашей сети → «Настроить DNS» → «Вручную».
  2. Сотовая сеть + Wi-Fi сразу: кнопка «Оптимизировать» выше — системный профиль на оба типа сети.
  3. После смены DNS перезапустите Wi-Fi или перезагрузите телефон.

FPS и производительность (1-10)

  • iOS 18+: включите «Игровой режим» — приоритизирует CPU/GPU игре и снижает задержку Bluetooth-контроллеров.
  • Закрывайте все фоновые приложения перед игрой (смахнуть вверх в App Switcher).
  • Настройки → Основные → Обновление контента — отключить фоновое обновление для всех, кроме мессенджеров.
  • Отключите Bluetooth, если не используете беспроводной контроллер/наушники.
  • Подключайтесь к Wi-Fi в диапазоне 5 ГГц, а не 2.4 ГГц.
  • Пункт управления → Фокусирование → создайте профиль «Игра» — глушит уведомления на время сессии.
  • Не держите телефон в чехле при долгой сессии — перегрев снижает частоту процессора.
  • Настройки → Батарея → отключите режим энергосбережения перед игрой.
  • Освобождайте не менее 10-20% памяти устройства — на заполненном хранилище система тормозит.
  • Для соревновательных игр — проводные наушники: задержка звука ниже, чем у Bluetooth.

Качество картинки и графика (11-20)

  • Настройки → Экран и яркость → True Tone — выключите для точной цветопередачи в играх.
  • Ночной режим (Night Shift) — выключайте на время игр, чтобы не искажались цвета.
  • Включайте HDR для видео, где он поддерживается экраном.
  • Специальные возможности → Движение → «Уменьшение движения» — ниже нагрузка на GPU, стабильнее FPS.
  • Специальные возможности → Дисплей → «Увеличить контрастность» — точнее тёмные сцены.
  • Специальные возможности → Дисплей → «Уменьшить прозрачность» — меньше нагрузки на композитор GPU.
  • Ручная яркость вместо авто — устраняет паразитные скачки экспозиции.
  • В настройках самой игры ищите Graphics/Frame Rate/Performance Mode — вручную снижайте, если греется.
  • Статичные обои вместо динамических — меньше фоновой GPU-анимации.
  • Отключите виджеты на экране блокировки, если не пользуетесь — они рендерятся постоянно.

Общее здоровье устройства (21-30)

  • Настройки → Основные → Обновление ПО — держите iOS актуальной.
  • Настройки → Основные → Хранилище iPhone — чистить раз в месяц, после 80% система тормозит.
  • Перезагружайте телефон каждые несколько дней — сбрасывает утечки памяти в фоне.
  • Настройки → Батарея → «Работоспособность аккумулятора» — при сильном износе iOS сама троттлит пик.
  • AirDrop/Handoff — выключить, если не используете: постоянное Bluetooth-сканирование грузит систему.
  • Удаляйте неиспользуемые приложения через «Выгрузить неиспользуемые приложения» в настройках хранилища.
  • Отключите автосинхронизацию Фото в iCloud на время игровых сессий — фон грузит CPU и сеть.
  • Проверьте геолокацию по приложениям — «Никогда» для тех, кому она не нужна, экономит батарею и CPU.
  • Держите заряд между 20% и 80% для долгосрочного здоровья батареи, если часто заряжаете от сети.
  • Избегайте прямого солнца и экстремальной жары/холода — они форсируют троттлинг производительности.

Установка на экран Домой

1. Нажми значок «Поделиться» внизу экрана Safari.

2. Выбери «На экран «Домой».

3. Нажми «Добавить» — иконка появится на главном экране и будет открываться как приложение.

Установка профиля DNS

Сейчас откроется Safari и скачается конфигурационный профиль iOS (.mobileconfig) с приватными DNS-резолверами. Он не является VPN и не туннелирует трафик — меняется только резолвинг DNS, системно на Wi-Fi и сотовой сети.

Установка происходит только через официальный системный экран iOS. После скачивания: Настройки → Профиль загружен → Установить → код-пароль → Установить ещё раз. Удалить можно в любой момент: Настройки → Основные → VPN и управление устройством → выбрать профиль → Удалить.
